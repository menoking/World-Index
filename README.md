# 🌍 世界指数 / World Index

基金板块力导向网络图，追踪热度指数，了解板块分布。

## 🎯 功能 / Function

* 🌐 世界网络

  * A股 / 指数 / QDII海外 / 债券
  * 板块力驱动网络图谱
  * 板块内基金净资金
  * 基金十大持仓
* 🔥 热度指数

  * 板块基金涨跌排名
  * 日排 / 周排
* 🌐 World Network

  * A-Shares / Index / QDII Overseas / Bonds
  * Sector force-driven network graph
  * Net fund flow within sectors
  * Top 10 fund holdings
* 🔥 Trending Index

  * Sector fund gain/loss rankings
  * Daily / Weekly rankings
